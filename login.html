LOGIN INTERFACE

Login Interface


This project was intended to have a login interface of a webpage. This includes the user email and password. This is very helpful to have access in your own account at a certain application.



There are various steps to make this. It inludes the following steps:

STEP 1: Make a html file and code your texts.
STEP 2: Make a css file and connect it to your html file. Start to code for the design.
STEP 3: Required your password and email by putting require in the nav bar.
FINAL STEP: Run and Test your code.
