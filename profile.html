Single Profile Homepage


This project was intended to have a single profie homepage in which you can discuss your personal information. This may helpful if you are aiming to build a webpage that can be accessible by people who are curios in your life.



There are various steps to make this. It inludes the following steps:

STEP 1: Make a html file and code your texts.
STEP 2: Make a css file and connect it to your html file. Start to code for the design.
STEP 3: Make a navigation link in the header for your profile pages.
FINAL STEP: Run and Test your code.
