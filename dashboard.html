DASHBOARD

Dashboard


This project was intended to have a dashboard just like our EVSU portal. These includes a three dashboard in the upper side of the webpage and a list of information below in which you can search a data by just typing its ID, Name, Email or Number.



There are various steps to make this. It inludes the following steps:

STEP 1: Make a html file and code your texts.
STEP 2: Make a css file and connect it to your html file. Start to code for the design.
STEP 3: Make a table and add 20 different data, and put a search bar.
FINAL STEP: Run and Test your code.
